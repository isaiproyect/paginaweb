Conceptos Solicitados

1.-¿Qué es un protocolo http?

-Un protocolo HTTP es un conjunto de reglas que define cómo se comunican los servidores y los navegadores web para transmitir datos de Internet.

2.-¿Qué es un Query String?

-Es un conjunto de parámetros que se pasan en una URL para solicitar datos específicos del servidor ( cadena de consultas).

3.-¿ Que es un verb HTTP?

-Los verbos HTTP son métodos de solicitud que indican la acción que se desea realizar en un recurso web.

4.-¿Qué es un DNS?

-DNS (Domain Name System)= sistema de nombres de dominio , es un sistema que se utiliza para traducir nombres de dominio legibles para los humanos en direcciones IP numéricas que los ordenadores pueden entender.

5.-¿Qué es un CDN?

-CDN (Content Delivery Network)=Red de entrega de contenidos es una red distribuida de servidores que se utilizan para entregar contenido web a los usuarios de la manera más rápida y eficiente posible.
