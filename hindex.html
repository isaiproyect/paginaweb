¿Qué es un protocolo http?

Un protocolo HTTP es un conjunto de reglas que define cómo se comunican los servidores y los navegadores web para transmitir datos a través de Internet.

¿Qué es un Query String?

Un Query String es una cadena de texto que se utiliza en una URL para enviar datos adicionales al servidor web, como parámetros de búsqueda o información de sesión.

¿Qué es un verbo HTTP?

un verbo HTTP

un método utilizado para solicitar una acción específica en un recurso web
